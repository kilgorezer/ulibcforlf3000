# glibc for LF3000 Version 2.x
While this library currently contains identical code to 1.x, it will be updated when 2.x goes in beta or release.\
\
glibc and a few commands ported to the built-in OS on LF3000 devices.\
This repository only contains the installer and readme as they are the only from-scratch components, however you can download the full binaries in releases.\
